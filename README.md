# SCL-90 症状自评量表

https://wenxideng-dot.github.io/scl-90/

基于 1986 年中国修订版标准开发的 **SCL-90 心理健康自评工具**。旨在为用户提供便捷、私密且交互友好的心理健康筛查服务。

## 📊 测评维度

本量表包含以下 10 个心理健康维度：

* 躯体化、强迫症状、人际敏感、抑郁、焦虑、敌对、恐怖、偏执、精神病性及其他（睡眠与饮食）。

## 🚀 快速开始

1. **在线访问**：通过 GitHub Pages 生成的链接直接进入测评。
2. **填写问卷**：根据过去一周的实际感受，点击对应的选项（无、轻、中、重、极重）。
3. **查看报告**：点击底部的“生成分析报告”，系统将自动计算总分、阳性项数及各因子分。
4. **保存结果**：如有需要，可点击“下载 CSV”保存详情。

## ⚖️ 声明

1. 本项目仅作为心理健康筛查的辅助工具，测评结果受受试者主观情绪、环境等因素影响。
2. **测评结果不代表临床诊断**。如测评数值偏高，仅提示可能存在相关症状，建议前往正规医疗机构咨询专业心理医生。
